# WeatherApp
A simple application that uses Retrofit and OpenWeatherMap API to check the weather by city name. 

# Techologies
* Kotlin 1.6.10
* Kotlin Coroutines 1.3.9
* Retrofit 2.9.0
* Gson converter 2.1.0
